# 🎬 Netflix Content Analytics Dashboard using Microsoft Power BI

## 📌 Project Overview

This project is an interactive Netflix Content Analytics Dashboard developed using Microsoft Power BI. It provides insights into Netflix's content library by analyzing movies, TV shows, genres, release trends, directors, countries, and content additions over time.

The dashboard enables users to explore Netflix's global catalog through interactive visualizations and filters.

---

## 🎯 Objectives

- Analyze Netflix movies and TV shows.
- Explore genre distribution.
- Study content release trends over time.
- Visualize country-wise content availability.
- Compare Movies vs TV Shows.
- Track content additions by month and year.

## 📈 Key Insights

- Movies represent a larger share of Netflix's catalog than TV Shows.
- Content additions increased rapidly after 2015.
- The United States contributes the largest number of Netflix titles.
- Drama and Documentary genres are among the most common.
- Content additions peaked around 2019–2020.
